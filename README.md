## Getting Started

## Setup

### Clone this repository

```bash
$ git clone https://github.com/fobadara/morse-decoder.git
$ cd morse-decoder
```


## Install linter

```bash
$  bundle init
$  bundle install
```

## Run linter

```bash
$  rubocop .
```

### Auto-correct

In auto-correct mode, RuboCop will try to automatically fix offenses:

```bash
$ rubocop -A
# or
$ rubocop --auto-correct-all
```
